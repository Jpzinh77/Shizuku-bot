##  Sobre o Projeto
A **Shizuku Bot Base** foi desenvolvida por **jpzinh**  para ser a porta de entrada de quem quer um bot rápido, leve e otimizado. Por ser uma **Base**, ela foca na estrutura principal, permitindo que vocę adicione seus próprios comandos e funcionalidades do seu jeito.

> [!IMPORTANT]
> Este projeto é distribuído gratuitamente. Por se tratar de uma base gratuita, **năo fornecemos suporte técnico para instalaçăo ou adiçăo de comandos**.

---

##  O que vem na Base? (Conteúdo Exclusivo)
Diferente de outras bases vazias, eu liberei um pack de funçőes prontas para vocę:

*  **Arquivo `SHIZUKU-CASES.js`**: Um pack exclusivo com diversas cases prontas!
*  **Customizaçăo Total**: Vocę pode pegar as cases desse arquivo e adicionar no seu bot do jeito que desejar.
*  **Agilidade**: Economize horas de programaçăo usando os modelos já testados e aprovados.

##  Configuraçăo de API (Importante)
Esta base **năo acompanha Key de API inclusa**. Para que as funçőes de consulta e ferramentas funcionem corretamente, vocę deve adquirir sua chave de acesso oficial.

### Como conseguir sua Key:
1. Acesse: [shizukuapis.space/planos](https://shizukuapis.space/planos)
2. Escolha o plano que melhor se adapta ao seu bot (Starter, Ouro ou Diamond).
3. Insira sua Key no arquivo de configuraçăo (`dono.json`) da base.
