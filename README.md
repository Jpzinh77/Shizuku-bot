##  Sobre o Projeto
A **Shizuku Bot Base** foi desenvolvida por **jpzinh**  para ser a porta de entrada de quem quer um bot rápido, leve e otimizado. Por ser uma **Base**, ela foca na estrutura principal, permitindo que você adicione seus próprios comandos e funcionalidades do seu jeito.

> [!IMPORTANT]
> Este projeto é distribuído gratuitamente. Por se tratar de uma base gratuita, **não fornecemos suporte técnico para instalação ou adição de comandos**.

---

##  O que vem na Base? (Conteúdo Exclusivo)
Diferente de outras bases vazias, eu liberei um pack de funções prontas para você:

*  **Arquivo `SHIZUKU-CASES.js`**: Um pack exclusivo com diversas cases prontas!
*  **Customização Total**: Você pode pegar as cases desse arquivo e adicionar no seu bot do jeito que desejar.
*  **Agilidade**: Economize horas de programação usando os modelos já testados e aprovados.

##  Configuração de API (Importante)
Esta base **não acompanha Key de API inclusa**. Para que as funções de consulta e ferramentas funcionem corretamente, você deve adquirir sua chave de acesso oficial.

### Como conseguir sua Key:
1. Acesse: [shizukuapis.space/planos](https://shizukuapis.space/planos)
2. Escolha o plano que melhor se adapta ao seu bot (Starter, Ouro ou Diamond).
3. Insira sua Key no arquivo de configuração (`dono.json`) da base.
